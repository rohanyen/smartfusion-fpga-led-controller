# SmartFusion FPGA Embedded System Project

FPGA and embedded systems implementation on Microchip SmartFusion SoC featuring Verilog-based hardware design, MSS programming in Embedded C, UART communication, APB interfacing, and FPGA simulation.

## Features
- Verilog HDL implementation
- FPGA waveform simulation
- UART communication using Tera Term
- APB bus interfacing
- MSS programming in Embedded C
- Hardware-software co-design

## Technologies Used
- Verilog HDL
- Embedded C
- SmartFusion SoC
- Libero SoC
- SoftConsole
- Tera Term

## Applications
- Aerospace embedded systems
- FPGA-based digital design
- Hardware acceleration systems
